# EchoID IP Character Designs

This folder contains the current low-poly MBTI-inspired IP character concepts for EchoID.

## Art Direction

- Original EchoID voice-personality avatars, inspired by personality-test character systems.
- Stylized 3D low-poly / geometric cartoon characters.
- Full-body, big-head small-body proportions.
- One symbolic prop per role.
- Clean pastel result-card composition.
- Designed to avoid copying any existing MBTI / 16Personalities character.

## Base Prompt Pattern

```text
Create an original MBTI-inspired character avatar for EchoID, but do not copy any existing 16Personalities character.

Visual style: stylized 3D low-poly personality-test avatar, full-body, big head and small body proportions, angular faceted clothing, simplified friendly face, matte material, crisp edges, clean silhouette, polished editorial quality.

Composition: centered full-body character, 3/4 front view, generous padding, vertical 4:5 result-card composition.

Brand adaptation: this is for EchoID, a voice-personality analysis product. The character should reflect speaking style, rhythm, emotion, and vocal presence through props, posture, and color.

Avoid: copying existing MBTI avatars, text, logo, watermark, realistic human face, anime style, cluttered background, excessive accessories.
```
